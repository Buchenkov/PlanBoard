Strt:
    .\venv\Scripts\Activate.ps1
    python -m app.main



Что это за файлы:
    - tasks.db — основной файл базы.
    - tasks.db-wal — журнал “опережающей” записи (куда сначала пишутся изменения).
    - tasks.db-shm — файл “shared memory” для координации процессов/потоков.

    Файлы -wal и -shm — часть нормальной работы SQLite в режиме WAL. 
    По завершении программы - сами удаляются.


Сборка:
    1) Подготовьте файлы иконок
    - Положите в app/resources/icons:
    - minimize.svg
    - maximize.svg
    - restore.svg
    - close.svg
    - app.ico (или app.png, 256x256)

2) Qt Resource (.qrc) и компиляция

<!DOCTYPE RCC><RCC version="1.0">
<qresource prefix="/icons">
    <file>icons/minimize.png</file>
    <file>icons/maximize.png</file>
    <file>icons/restore.png</file>
    <file>icons/close.png</file>
    <file>icons/app.ico</file>
</qresource>
</RCC>


Сохраните как app/resources.qrc и скомпилируйте:

    pyrcc5 -o app/resources_rc.py app/resources.qrc



3) Импорт ресурсов и установка иконок
    - В начале views.py добавьте:

    from app import resources_rc  # регистрирует ресурсы :/icons/...
                                    или полный адрес

EXE

упакуем текущий проект в exe для Windows 7 (Python 3.7 x86) с PyInstaller. 
Для Python 3.7-32, рекомендуется PyInstaller 4.10 — он стабильно работает на Win7 и поддерживает 3.7.

Шаги установки в venv:

- python -m pip install --upgrade pip
- pip uninstall -y pyinstaller
- pip install "pyinstaller==4.10"


Собрать проект.  Если есть build.bat

    .\build.bat


Если build.bat нет или хочешь вручную — выполни команду PyInstaller:

pyinstaller --noconfirm --name "PlanBoard" --onefile --windowed ^
  --icon "app\resources\icons\app.ico" ^
  --add-data "app\resources;app\resources" ^
  app\main.py


5) Проверь результат

dir dist
.\dist\PlanBoard.exe
